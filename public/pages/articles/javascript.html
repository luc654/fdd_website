Javascript

Created: 28 Jan, 2026

JavaScript is een scripttaal waarmee je webpagina’s interactief kan maken. Een stukje van een scripttaal bestaat uit een aantal opdrachten die de browser uitvoert. De browser ontvangt deze opdrachten als een HTML-pagina wordt ingeladen. De scripttaal staat namelijk in de HTML-code. Dat kan op twee manieren:
  • direct in de code via <script>-tags;
  • met een verwijzing naar externe code.

JavaScript is dus een veelzijdige scripttaal die webpagina's interactief en dynamisch maakt. Het is een van de drie kerntechnologieën van het web (naast HTML en CSS), waarmee functionaliteiten zoals animaties, pop-ups, formulieren en interactieve kaarten worden toegevoegd. JavaScript wordt uitgevoerd in de browser (client-side) van de gebruiker, dit betekent dat de gebruiker alles kan inzien en aanpassen, mits ze de juiste kennis hebben.

Kan ik JavaScript?

Het mooie van JavaScript is de laagdrempeligheid en het brede gebruik van JavaScript. Je kan JavaScript NU gaan coderen in je terminal, druk op deze toetsen om de console te openen in chrome: ctrl + shift + J. Type hierin alert(“Hallo!”); en druk op enter, je hebt zojuist javascript laten draaien. JavaScript is de enige echte programmeertaal van de kerntechnologieën HTML, CSS en JS. JS is de afkorting voor JavaScript, JavaScript bestanden worden ook opgeslagen als .js. Met dit kleine, onschuldige stukje code kan je ELKE webbrowser laten crashen. Dit lukt je niet met HTML of CSS

txt = "a";
while(1){
txt = txt += "a";
}

Met JavaScript kan je bijna alles, te veel volgens sommige mensen, maar die mensen snappen de kracht van JavaScript niet. Je kan met JavaScript bijvoorbeeld een server starten, dit doe je via NodeJS. Of je kan mobiele apps maken via het Framework React Native. Je kan zelfs google extensies schrijven via JavaScript.

Javascript op deze website

Natuurlijk wordt er ook JavaScript op deze website gebruikt. De knop naast de titel waarmee je de tekst groter kan maken is gemaakt met JavaScript, maar ook zwaardere functies zoals de zoek- en filter functies op de artikelpagina zijn beide met javascript gemaakt. De carrousel op de homepagina voor mobiel is met een JavaScript Library gemaakt (code van iemand anders) en de contact pagina wordt via JavaScript naar een server gestuurd! Javascript is dus een enorm belangrijke taal en ik zou niet zonder kunnen.
